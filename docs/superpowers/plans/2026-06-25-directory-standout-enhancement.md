# Directory Standout Enhancement Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Upgrade the directory with student decision-support fields, transparent ranking copy, filter chips, and stronger card information architecture.

**Architecture:** Keep the existing App Router route and in-memory data. Extend the `Practitioner` type, centralize ranking in the data module, let the client directory own selected specialism state, and keep the card presentational with a `matchesSelectedSpecialism` prop.

**Tech Stack:** Next.js 16 App Router, React 19, TypeScript, Tailwind CSS 4.
